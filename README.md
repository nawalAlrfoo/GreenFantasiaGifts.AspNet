# Green Fantasia Gifts

Green Fantasia Gifts is a specialized website designed to offer plants as meaningful and expressive gifts.

The site categorizes plants based on different occasions and emotions, allowing users to choose a plant that reflects feelings such as joy, apology, congratulations, or support. This approach adds a personalized and emotional touch to the gifting experience.

## Features
- ASP.NET Core MVC project
- Clean architecture and organized structure
- Data validation using DataAnnotations
- Responsive UI

## Notes
- Database and connection strings are excluded for security reasons.
